# whoa

网络不好时，加载数据会比较慢。实际项目中，启动app后，可开启后台下载任务，下载数据到本地，然后UI侧从本地读取数据。

备注：server端不提供图片宽高，则app端无论什么解决方案都不完美。并且即使提供图片宽高，瀑布流这种设计本身就吃内存，性能再优化也不会太好，故而当前极少有主流app采用这种技术，此工程解决方案意义不大，废弃。
